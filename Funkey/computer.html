Connecting to your Computer

Attach the USB to your computer and FunKey.

Press the ⏻ Menu button to open the settings.

Scroll ▲ Up to the Mount USB option and press  A  twice to mount.

You should see something along the lines of 'Detachable Drive' or 'USB Drive' appear connected to your computer.

Once finished, press  A  twice to dismount the drive.

Adding ROMS:

Locate the folder of the system your game is for. Our example will be Super Mario World, so you'd find the SNES folder.

Paste the ROM into the folder and name it whatever you'd like to appear in the menu.

If you want to add a cover image, you'll need to place a .PNG file with the same name in the same folder as the ROM.

For example, Super Mario World.sfc and Super Mario World.png would be in the same folder.

Once that's done, you can eject the drive and your ROMs should be there.

If not, check here to troubleshoot.

Adding Themes:

Adding a theme is different depending on whether you're using Gmenu2x or RetroFE.

Path for RetroFE: /FunKey/.retrofe/layouts/

Path for Gmenu2x: /FunKey/.gmenu2x/skins/240x240/

You paste your theme's folder into one of the folders above (depending on what kind it is).

In RetroFE, you have to change layouts.list in the layouts folder to include the name of the folder you added.

For example, if the theme folder was "RetroRoom", you'd add "RetroRoom" to the text file.


To apply a theme, read here.


Changing the GameBoy palette is the hardest. Go to /FunKey/./ (If it doesn't exist, make it.)

Make a new file called |, and inside write |.

Adding PS1 and GBA BIOSes:

PS1 and GBA games typically require a BIOS file to work.

For legal reasons, I can't really tell you where to get them.

I'm sure you can find them somewhere, though.

Just place the .bin file inside the bios folder inside the console's folder.
