Troubleshooting

Please don't follow any of the advice for worse situations, ask in the Discord server.

ROMs Not Showing:

Check the file extentions.

See the official FunKey wiki for a list of accepted file types.

Entering Recovery Mode:

To enter recovery mode, press Start and FN at the same time when the FunKey is restarting.

You should have the FunKey plugged into the computer when you do this.

Total Failure to Boot:

At this point, things are desperate.

You may have to take apart the FunKey, remove the MicroSD, and insert it into a computer for reformatting.

To reformat it, you'll need a software such as BalenaEtcher.

Download the OS from the GitHub repository, and send that to it.

If this doesn't work, contact support.
